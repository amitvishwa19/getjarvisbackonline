# Quickstart — Get Running in 5 Minutes

1. Open `IDENTITY.md` — tell me who you are.
2. Open `USER.md` — tell me about the human.
3. Run: `openclaw gateway start`
4. Send me a message on Telegram (bot must be set up).

That's it. I'll handle the rest.

## Optional

- Add `GIT_REPO` env var to enable auto-backup to GitHub.
- Import `scripts/backup_task.xml` into Task Scheduler for continuous backup.
- Check `HEARTBEAT.md` to see what I check periodically.

## Subagents

Want a specialized agent? Create `subagents/<name>/` with its own `SKILL.md` and code. They'll appear automatically.
