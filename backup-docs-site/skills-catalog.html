🧰 Capabilities • Expandable Skill Set

JARVIS Skills Catalog

Comprehensive list of capabilities that can be added as OpenClaw skills to extend JARVIS functionality.

📌 Each skill can be implemented as a separate module with its own API integrations or local scripts.

📧 Email & Communication

email-parser

Google Workspace

Parse incoming emails, extract content, attachments, metadata. Summarize unread messages.

Example: "Check unread emails from boss, summarize"

email-sender

Google Workspace

Compose & send emails via Gmail/Outlook/IMAP. Attach files, format HTML.

Example: "Send follow-up email to client"

email-responder

AI-Powered

AI-generated replies based on email context. Learn from past responses.

Example: "Auto-respond to meeting invites"

calendly-manager

Scheduling

Manage Calendly events, scheduling links. Book/cancel/reschedule.

Example: "Check my Calendly availability for tomorrow"

📅 Calendar & Scheduling

calendar-ai

AI-Powered

Smart scheduling, conflict resolution, optimal meeting times. Integrates with Google Calendar.

Example: "Schedule a 1-hour meeting with team next week"

reminder-manager

Utility

Set, track, follow up on reminders. Recurring patterns. Escalation logic.

Example: "Remind me every Friday to submit report"

timezone-converter

Utility

Multi-timezone meeting scheduler. Finds overlapping work hours across regions.

Example: "Find overlapping hours between India & US"

event-analyzer

Analytics

Analyze calendar patterns, suggest optimizations. Identify bottlenecks.

Example: "Why am I overbooked on Mondays?"

💼 Social Media & Marketing

linkedin-poster

LinkedIn

Post updates, articles, job listings to LinkedIn. Tag people, add hashtags.

Example: "Post a job opening for senior dev"

twitter-tweeter

Twitter/X

Tweet, create threads, retweet, follow, like. Handle mentions.

Example: "Tweet about latest AI breakthrough"

content-curator

RSS/Feeds

Find & schedule content from RSS feeds, news sources. Auto-post queue.

Example: "Post 5 tweets this week about AI trends"

social-analytics

Analytics

Track engagement, followers, mentions, sentiment. Weekly reports.

Example: "How did my last post perform?"

📊 Data & Analytics

google-analytics

Analytics

Fetch website stats, traffic reports, user behavior from GA4.

Example: "Show me last week's website traffic"

sheets-analyzer

Google Sheets

Read/write Google Sheets, analyze data, generate summaries, pivot tables.

Example: "Update sheet with today's metrics"

excel-processor

Local

Process .xlsx files locally using Python (openpyxl, pandas). No API needed.

Example: "Convert Excel report to JSON"

chart-generator

Visualization

Auto-generate charts (bar, line, pie) from data. Export as PNG/SVG.

Example: "Create bar chart of sales by region"

🔍 Research & Monitoring

arxiv-scraper

Research

Fetch latest research papers from arXiv.org by category, keyword, author.

Example: "Find latest papers on LLM quantization"

github-tracker

DevOps

Monitor GitHub repos: issues, PRs, commits, releases. Get notifications.

Example: "Any new issues in our repo?"

patent-searcher

Legal

Search patent databases (USPTO, EPO) for keywords, assignees, IPC codes.

Example: "Search patents for 'autonomous vehicles'"

news-monitor

RSS

Monitor news sources, RSS feeds, blogs. Keyword alerts, sentiment analysis.

Example: "Has there been news about OpenAI today?"

📁 Document & File Processing

pdf-extractor

OCR

Extract text from PDFs (scanned or digital). Preserve formatting, tables.

Example: "Read this contract PDF, summarize key terms"

image-ocr

OCR

Extract text from images using Tesseract. Supports multiple languages.

Example: "Read text from this screenshot"

doc-converter

Conversion

Convert between document formats: .docx → .pdf, .md → .html, etc.

Example: "Convert this .docx to PDF"

file-organizer

Local

Auto-organize files by type, date, project. Move, rename, archive.

Example: "Sort downloads folder by file type"
