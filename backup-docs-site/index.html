💾 Backup System

Configuration

  • • Repo: backup-system/backup-repo/
  • • Schedule: Daily at 2 AM
  • • Retention: 365 commits OR 90 days
  • • Compression: gzip -1 (fastest)
  • • Locking: .backup.lock/ (atomic)

Exclusions

  • • backup-repo, test-site
  • • node_modules, logs
  • • *.log, *.tmp
  • • .DS_Store, Thumbs.db

Watchdog (Auto-Recovery)

Runs every 5 minutes:

  • Check gateway health (http://127.0.0.1:18789/health)
  • Check last backup age (< 24h required)
  • If failed: Restore latest backup + restart gateway, log action
